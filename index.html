    📖 Cara Kerja File → Google Drive

    1️⃣
    User upload file → Dikonversi jadi Base64
    2️⃣
    Data + Base64 dikirim ke Google Apps Script
    3️⃣

    Status otomatis update di aplikasi (real-time listener)

    ⚠️ INFORMASI:

    • Maksimal 30MB per file (optimal untuk base64 encoding)
    • Format: PDF, JPG, PNG, DOC, XLS, PPT, ZIP
    • File muncul di Drive dalam hitungan detik (real-time processing)
    • Tidak perlu kartu kredit!
    • Status sync otomatis ter-update via real-time listener
